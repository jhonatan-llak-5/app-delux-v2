# Imágenes locales

Coloca aquí tus imágenes propias (zapatillas, productos, banners).

Formatos recomendados:
- WebP para fotos (mejor compresión)
- SVG para iconos vectoriales
- PNG transparente para productos sin fondo

Nomenclatura sugerida:
- products/sneaker-ultra-boost-light.webp
- products/sneaker-samba-og.webp
- banners/drop-01.webp

En el código se referencia como:
  src="assets/images/products/sneaker-ultra-boost-light.webp"
